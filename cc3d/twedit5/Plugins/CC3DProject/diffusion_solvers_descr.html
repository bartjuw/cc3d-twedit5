Diffusion solvers:

DiffusionSolverFE (default)

This solver is used when you do not know which one you want. Can handle secretion within volumes. Multiple secretion and diffusion calls are interleaved for a more gradual secretion then using the FlexibleDiffusionSolverFE. More info.

ReactionDiffusionFVM

The reaction diffusion finite volume (RDFVM) solver uses the finite volume method to solve a system of reaction diffusion equations. More info.

ReactionDiffusion

This is an older reaction diffusion solver. More info.

FlexibleDiffusionSolverFE

Another diffusion solver that can handle secretion. All the secretion amount is calculated once followed by possibly multiple diffusion calls (calculations). More info

FastDiffusionSolver2DFE

Two dimension specific diffusion solver

KernalDiffusionSolver

Another solver, useful when large diffusion constants are needed. Requires periodic boundaries. More info.

SteadyStateDiffusionSolver

Can be used when diffusion equations reach steady-state. Useful when diffusion constants are large. More info.
